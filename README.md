# Order Client

Projeto React + TypeScript + Tailwind CSS + shadcn/ui

## Setup

```bash
npm install
```

## Desenvolvimento

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Lint

```bash
npm run lint
```

## Estrutura

```
src/
├── components/
│   └── ui/           # Componentes do shadcn/ui
├── App.tsx           # Componente principal
├── index.css         # Estilos globais (Tailwind)
└── main.tsx          # Entry point
```

## Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- Lucide React
- ESLint + Prettier
